安装 / 配置实测记录

✅ 键盘输入 · 已解决

问题点了输入框敲键盘没反应
原因config.ini 里 hw.keyboard=no，模拟器以为没键盘
修复F:\1\avd\android-35.avd\config.ini 中 hw.keyboard=yes
备注v36 已移除 UI 里的键盘开关，只能改配置文件

✅ 网络 / DNS · 已解决

问题开机后浏览器打不开网页
原因DNS 为空，域名无法解析
修复启动脚本加 -dns-server 8.8.8.8,114.114.114.114

✅ "未连接网络"提示 · 已解决

原因Android 连不上 Google 服务器就显示未连接，实际网络正常
修复adb shell settings put global captive_portal_mode 0

✅ C 盘空间不足 · 已解决

问题AVD 默认写 C:\Users\...，userdata 需要 ~8GB
修复设 ANDROID_AVD_HOME=F:\1\avd 并重建 AVD
补充config.ini 里 disk.dataPartition.path 需改成相对路径

常用 ADB 命令速查

基础控制

查看设备adb devices
输入文字adb shell input text "hello"
按回车键adb shell input keyevent 66
返回键adb shell input keyevent 4
主页键adb shell input keyevent 3
锁屏adb shell input keyevent 26
截图到电脑adb exec-out screencap -p > screen.png
安装 APKadb install xxx.apk
杀模拟器adb emu kill

虚拟定位

定到经纬度adb emu geo fix <经度> <纬度>

网络诊断

ping 测试adb shell ping -c 2 baidu.com
检查 DNSadb shell getprop net.dns1
关掉未连接提示adb shell settings put global captive_portal_mode 0

语言切换

设为中文adb shell settings put system system_locales zh-CN
重启生效adb reboot
